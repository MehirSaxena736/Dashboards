<h1 align="left">From Crisis to Transformation</h1>

<p align="left">
An interactive Power BI dashboard analyzing the impact of COVID-19 on remote work adoption, employee well-being, productivity, and workplace transformation.
</p>

---

## Overview

**From Crisis to Transformation** explores how organizations adapted to remote work during and after the COVID-19 pandemic. The dashboard provides insights into employee well-being, work-life balance, productivity, and remote work trends through interactive visualizations.

---

## Key Features

- Remote Work Adoption Analysis
- Employee Well-Being Metrics
- Work-Life Balance Insights
- Productivity Analysis
- Interactive Filters and Slicers
- Dynamic Visualizations

---

## Technologies Used

- Power BI
- Microsoft Excel / CSV
- Data Visualization

## Project Goal

The objective of this project is to analyze the transformation of workplace practices during the COVID-19 pandemic using interactive Power BI dashboards. It enables users to explore workforce trends and employee well-being through meaningful visual insights.
